Hold R : Rotate camera
Hold R + wheel : Zoom camera
Hold M : Move camera xy axis
Hold M + wheel : Move camera z axis
Press H : Reset camera
Hold X/Y/Z + vertex table active : Move vertex coordinate
Hold R/H/X/Y/Z + vertex table active + click : Select vertex near camera point

Icon Delete harus double click
Save file berformat binary (format buatanku),
Belum support (e.g. OBJ, FBX, 3DS, PLY, STL, etc.)
Sumbu XYZ masingmasing berukuran 1000 units

menggunakan animasi (belum sedia editor animasi, hanya bisa pake js)
